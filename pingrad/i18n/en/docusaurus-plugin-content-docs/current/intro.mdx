# Welcome to PIN Grad

PIN Grad is a practical documentation hub built by PIN students for graduate applicants.

## Why this exists

- Turn scattered experience into reusable knowledge.
- Focus on methods and execution, not outcome-only stories.
- Reduce trial-and-error time for new applicants.

## What is different

Compared with generic portals, PIN Grad is more aligned with real PIN project workflows:

- Unified data fields.
- Process-first tracking.
- Outcome plus long-term path review.

## How to contribute

Open a PR or issue in GitHub:

- Fix outdated info.
- Add templates and examples.
- Submit reproducible case notes.
