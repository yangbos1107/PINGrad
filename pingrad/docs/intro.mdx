# 欢迎来到 PIN Grad

PIN Grad 是一个围绕 PIN 项目成员构建的研究生申请与发展知识站。它不追求“大而全”的百科式收录，而强调可执行、可复用、可追踪的经验沉淀。

## 我们做这个的动机

- 将分散在聊天记录、表格、个人笔记里的经验结构化。
- 把“成功案例”拆成可复用的方法，而不是结果展示。
- 让后来者用更低时间成本建立自己的申请策略。

## 与其他站点的差异

PIN Grad 更贴近 PIN 项目的真实场景：

- 数据口径统一：使用同一套字段描述背景、目标与结果。
- 强调过程跟踪：不仅看 offer，还看每一步的决策逻辑。
- 关注长期结果：入学只是开始，后续成长路径同样被记录。

## 你可以从哪里开始

1. 浏览 [DataPoints 总览](/docs/datapoints/intro) 了解数据结构。
2. 进入 [申请跟踪](/docs/tracking/intro) 使用流程模板。
3. 按 [PIN 梯度导航](/docs/S) 找到与你最接近的实践路径。

## 如何贡献

欢迎通过 PR 或 Issue 补充内容：

- PR：修正文案、补充样例、更新模板。
- Issue：报告过期信息、提出结构建议。
- 内容提交：可先用表单（占位）提交，再由维护者整理入库。
