# B+ 梯度总览

B+ 档是稳健主力档，用于提高整体录取确定性。

- [The Pennsylvania State University](/docs/b-plus/the-pennsylvania-state-university) — M.S. Electrical Engineering
- [Texas A&M University, College Station](/docs/b-plus/texas-a-and-m-university-college-station) — MS, Electrical Engineering

- [University of California, Santa Barbara](/docs/b-plus/university-of-california-santa-barbara) — MS: Electrical and Computer Engineering

## 建议策略

1. 优先保证材料完整度和提交时效。
2. 结合项目方向强调工程落地能力。
3. 与 A/A- 档共用基础素材，降低维护成本。

## 使用方式

- 左侧 Sidebar 可直接展开该梯度并进入具体学校项目页。
- 每个学校页统一包含：项目介绍、录取bar和dp、RA、找工、生活。
