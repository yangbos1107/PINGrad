# 去向数据

去向数据关注的是“结果之后”的路径，而不是仅停留在 offer 数量。

## 核心指标

| 指标 | 说明 | 示例 |
| --- | --- | --- |
| Destination Type | 去向类型 | PhD / Master / Industry |
| Region | 地区 | US / EU / APAC |
| Role | 角色 | RA / SWE / DS |
| Transition Window | 转换周期 | 入学后 6-18 个月 |

## 解读原则

- 优先看样本数量，再看比例。
- 与同梯度人群比较，避免跨背景误读。
- 关注路径连续性，例如“硕士 -> 博士”衔接率。

## 常见误区

- 用单年数据做长期结论。
- 只展示最优结果，忽视中位路径。
- 忽略地区和项目类型差异。
