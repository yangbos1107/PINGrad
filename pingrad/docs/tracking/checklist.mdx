# 申请检查清单

## 每周固定检查

- [ ] 截止日期是否有变化
- [ ] 文书版本是否同步
- [ ] 推荐信状态是否确认
- [ ] 网申系统状态是否记录

## 节点前检查（截止前 72 小时）

- [ ] PDF 导出和排版一致
- [ ] 项目名称与专业方向无误
- [ ] 联系方式和成绩单版本正确
- [ ] 申请费支付状态正常

## 结果后检查

- [ ] 记录 decision 与日期
- [ ] 记录关键反馈信息
- [ ] 更新个人策略与下一步行动
